# Edita

A modern text editor similar to Notepad++, built with vanilla JavaScript. Works as both a web app and installable desktop application (PWA).

## Features
- Multiple tabs support
- Light/Dark theme toggle
- Find and replace functionality
- Open and save files
- Comprehensive error logging
- Install as desktop app (PWA)
- Keyboard shortcuts (Ctrl+S, Ctrl+O, Ctrl+F, Ctrl+H, Ctrl+N)
- Line numbers and status bar

## Usage
Open index.html in a browser or visit the GitHub Pages link.
